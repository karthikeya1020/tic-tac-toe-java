# Tic Tac Toe (Java)
A simple console-based Tic Tac Toe game built using Java. This project allows two players to play the game in the terminal with automatic win and draw detection.

## Features
- Two-player gameplay
- Win detection (rows, columns, diagonals)
- Draw Detection
- Prevents overwriting occupied cells
- Console-based interface

## How to Run
'''bash
javac TicTacToe.java
java TicTacToe
'''

## Technologies Used
- Java
- OOP
- Arrays
- Scanner
